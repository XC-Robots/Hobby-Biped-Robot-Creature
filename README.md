# Hobby-Biped-Robot-Creature
This is a biped robot I've been working on. It has two legs each with 4DOF and a moving eye with eyelids. The purpose of this build is to familiarize myself with inverse kinematics, proportions, electronic use, and code, as well as to have a fun little robot walking around. I used onshape to cad this because it was the only cad software I knew at the time.
