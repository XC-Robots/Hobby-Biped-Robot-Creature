# Hobby-Biped-Robot-Creature
This is a biped robot I've been working on. It has two legs each with 4DOF and a moving eye with eyelids. The purpose of this build is to familiarize myself with inverse kinematics, proportions, electronic use, and code, as well as to have a fun little robot walking around. I used onshape to cad this because it was the only cad software I knew at the time. This robot is unique in its design as it focuses on character, with special details such as the eye and mouth, without sacrificing functionality. All the parts, save the electronics and servos, were 3d-printed, and everything is mounted together using M3 screws. The bot uses a total of 12 servos, 4 per leg, 2 for the eye, one for the eyelid, and one for the mouth.
